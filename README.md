## Nami
### A lightweight utility to control mouse movement entirely via keyboard (Linux X11). 
### Keybindings
| Shortcut | Action |
| :--- | :--- |
| **Caps Lock** | Enable Nami |
| **Caps + W** | Move Up |
| **Caps + X** | Move Down |
| **Caps + A** | Move Left |
| **Caps + D** | Move Right |
| **Caps + [Combo]** | Diagonal Movement (e.g., W+D for Up-Right) |
| **Caps + J (Hold)** | Left Click / Mouse Down |
### Installation
```bash
# Navigate to the directory
cd Nami

# Run with sudo
sudo ./Nami
